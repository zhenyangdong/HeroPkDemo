伤害计算公式梳理

1) 命中判定：hitChance = clamp(0.72 + (命中率 - 闪避率) × 0.20, 0.35, 0.99)

2) 随机攻击：rolledAttack = attackValue × (1 + ((rand × 0.16) - 0.08) × randomScale)

3) 防御减伤：mitigation = defenseValue / (defenseValue + 260)

4) 基础伤害：base = max(rolledAttack × (1 - mitigation × 0.90), attackValue × minDamageRatio)

5) 暴击修正：critChance = clamp(暴击率 - 格挡率 × 0.25, 0.02, 0.55)，触发后 base ×= clamp(暴伤倍率, 1.05, 1.75)

6) 格挡修正：blockChance = clamp(格挡率 - 命中率 × 0.08 + 0.02, 0.03, 0.50)

7) 封顶落地：damage = round(min(base, 目标最大生命 × capRatio))，最小为 1（未命中则 0）

示例（普通攻击）

设：attackValue=185，defenseValue=155，randomScale=0.55，minDamageRatio=0.20，capRatio=0.24，目标 maxHp=1000。

命中率 98%，闪避率 16%：hitChance = 0.72 + (0.98 - 0.16) × 0.20 = 0.884（88.4%）

若本次随机因子取 0.98：rolledAttack = 185 × 0.98 = 181.3

mitigation = 155 / (155 + 260) = 0.373

base = 181.3 × (1 - 0.373 × 0.90) = 120.5（高于保底 185 × 0.20 = 37）

若未暴击且未格挡，封顶为 min(120.5, 1000 × 0.24=240)，最终伤害约为 121。
